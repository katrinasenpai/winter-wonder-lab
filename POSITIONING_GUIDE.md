# Руководство по привязке элементов к фону

## Проблема
Фон использует `background-size: cover`, что масштабирует его независимо от элементов. Элементы позиционированы относительно viewport, а не фона.

## Решение 1: Использовать единый масштабируемый контейнер (РЕКОМЕНДУЕТСЯ)

### Шаг 1: Определить исходный размер фона
Узнай размер исходного изображения `lab-background.png` (например, 1920×1080px)

### Шаг 2: Изменить CSS для фона
Замени `background-size: cover` на фиксированное соотношение сторон:

```css
.background {
    background-size: contain; /* или конкретный размер */
    background-position: center top; /* или нужная позиция */
    aspect-ratio: 16/9; /* соотношение сторон твоего фона */
}
```

### Шаг 3: Использовать CSS переменные для масштаба
Добавь в `:root`:

```css
:root {
    --bg-width: 1920px;  /* ширина исходного фона */
    --bg-height: 1080px; /* высота исходного фона */
    --scale: min(100vw / var(--bg-width), 100vh / var(--bg-height));
}
```

### Шаг 4: Применить масштаб к контейнеру
```css
.lab-container {
    transform: scale(var(--scale));
    transform-origin: top left;
    width: var(--bg-width);
    height: var(--bg-height);
}
```

## Решение 2: Использовать viewport units с учетом соотношения сторон (БЫСТРЕЕ)

Если фон 1920×1080 (16:9), используй:

```css
/* Все позиции в процентах от viewport, но с учетом соотношения */
.element {
    left: calc(50% - (100vw * 0.5)); /* пример */
    top: calc(10vh * (16/9) / (100vw / 100vh)); /* для сохранения пропорций */
}
```

## Решение 3: Использовать JavaScript для автоматической привязки (САМОЕ БЫСТРОЕ)

Создай скрипт, который:
1. Загружает изображение фона
2. Определяет его размер
3. Вычисляет масштаб
4. Применяет transform: scale() к контейнеру

## Быстрый способ: Использовать DevTools

1. Открой DevTools (F12)
2. Выбери элемент в Elements
3. Измени позиции в реальном времени
4. Копируй значения в CSS

## Рекомендация

Используй **Решение 1** с фиксированным соотношением сторон - это самый надежный способ.
