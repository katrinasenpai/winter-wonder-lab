# Спецификация изображений для "Зимняя Лаборатория Чудес"

## Общие требования
- **Формат**: PNG с прозрачным фоном (для всех элементов, кроме фона)
- **Разрешение**: 2x (Retina) для чёткости на экранах высокого разрешения
- **Стиль**: Cozy 2D, рисованная графика, тёплые тона
- **Цветовая палитра**: Глубокие синие/фиолетовые для фона, тёплые коричневые/янтарные для элементов

---

## 1. ФОНОВЫЕ ИЗОБРАЖЕНИЯ

### 1.1. Стена (wall-background.png)
- **Размер**: 1920×1080px (или больше, будет масштабироваться)
- **Формат**: PNG или JPG (без прозрачности)
- **Описание**: Тёмно-синяя/фиолетовая стена с текстурой, возможно с полками на заднем плане
- **Использование**: `.wall` background-image

### 1.2. Окно (window-frame.png)
- **Размер**: ~600×500px (для окна 35% ширины экрана)
- **Формат**: PNG с прозрачностью
- **Описание**: Рама окна в тёмно-коричневом цвете, возможно с морозными узорами на стекле
- **Использование**: `.window` background-image или border-image

### 1.3. Полки (shelves.png)
- **Размер**: ~300×400px
- **Формат**: PNG с прозрачностью
- **Описание**: Три горизонтальные полки, тёмно-коричневые, возможно с книгами/склянками
- **Использование**: `.shelves` background-image

### 1.4. Стол (desk-surface.png)
- **Размер**: 1920×600px (широкий, для нижней части)
- **Формат**: PNG или JPG
- **Описание**: Деревянная поверхность стола с текстурой, тёплый коричневый цвет
- **Использование**: `.desk` background-image

### Оптимизация:
1. Используй WebP для больших фоновых изображений (с fallback на PNG)
2. Сжимай изображения через TinyPNG или ImageOptim
3. Для Retina-экранов создавай версии @2x (удвоенное разрешение)
4. Используй SVG для простых иконок (если возможно)

### Приоритет создания:
1. **Высокий**: Фон стены, стол, окно, планшет с картой, монитор, колба
2. **Средний**: Лампа, доска с карточками, декоративные элементы
3. **Низкий**: Анимации, спрайты, дополнительные детали
