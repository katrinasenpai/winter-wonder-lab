# Изображения для карточек мероприятий

## Папка для изображений
Все изображения нужно поместить в папку: **`assets/images/`**

## Размер изображений
**Рекомендуемый размер: 280px × 360px** (соотношение сторон примерно 1:1.3)

Это обеспечит хорошее качество на карточках размером 140px × 180px (изображение будет масштабироваться с помощью `background-size: cover`).

Можно использовать изображения большего размера (например, 560px × 720px для Retina-дисплеев), но минимальный размер должен быть не менее 280px × 360px.

## Формат
**Формат: PNG** (с прозрачностью или без, по желанию)

## Примечания
- Изображения будут автоматически масштабироваться и обрезаться по центру (`background-size: cover`)
- Рекомендуется использовать изображения с хорошим контрастом, чтобы текст на карточках был читаемым
- Изображения должны соответствовать тематике каждого мероприятия
