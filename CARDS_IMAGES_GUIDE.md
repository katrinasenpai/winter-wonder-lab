# Руководство по созданию изображений для карточек мероприятий

## Общие требования

- **Размер:** 400×300px (соотношение 4:3)
- **Формат:** JPG или PNG
- **Стиль:** Cozy 2D, новогодняя тематика, тёплые тона
- **Качество:** Высокое разрешение для чёткости

## Стилистические рекомендации

1. **Единый стиль:** Все карточки должны быть в одном стиле (cozy 2D, рисованная графика)
2. **Новогодняя тема:** Добавь зимние/новогодние элементы (снежинки, гирлянды, свечи)
3. **Тёплая атмосфера:** Используй мягкие, тёплые цвета
4. **Детали:** Карточки должны быть узнаваемыми и привлекательными
5. **Композиция:** Центральный элемент, окружённый декоративными деталями

## Технические детали

- **Разрешение:** Минимум 400×300px, лучше 800×600px для Retina
- **Сжатие:** Оптимизируй изображения (TinyPNG, ImageOptim)
- **Формат:** JPG для фотографий, PNG для графики с прозрачностью
- **Вес файла:** Старайся уложиться в 100-200KB на изображение

## Примеры композиций

### Для поэзии:
Книга, открытая на странице со стихами, рядом перо и чернильница, на заднем плане — зимний пейзаж за окном, свеча

### Для концерта:
Фортепиано с открытой крышкой, ноты на пюпитре, новогодние гирлянды вокруг, праздничная атмосфера

### Для видеокриейторов:
Камера на штативе, экран с интерфейсом монтажа, киноплёнка, современная творческая студия

### Для лекции о районе:
Старая карта района, старинные фотографии, компас, лупа, уютная библиотечная атмосфера

### Для мастер-класса:
Кисти, краски, материалы для творчества, магические символы, сказочные элементы

### Для Толкина:
Кольцо на открытой книге, карта Средиземья, старинные свитки, мистическая атмосфера
