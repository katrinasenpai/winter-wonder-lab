# Быстрая настройка позиций элементов

## Способ 1: Использовать скрипт position-helper.js (САМЫЙ БЫСТРЫЙ)

1. Открой сайт в браузере
2. Открой консоль (F12 → Console)
3. Введи: `positionHelper.start()`
4. Кликай на элементы, которые нужно позиционировать
5. В консоли появятся точные значения позиций
6. Копируй значения в CSS

**Остановить:** `positionHelper.stop()`

**Установить размер фона:** `positionHelper.setBgSize(1920, 1080)`

---

## Способ 2: Использовать DevTools (ВИЗУАЛЬНО)

1. Открой DevTools (F12)
2. Выбери элемент в Elements
3. В Styles измени `left`, `top`, `width`, `height`
4. Смотри результат в реальном времени
5. Копируй значения в CSS

**Совет:** Используй проценты (%) вместо пикселей для адаптивности

---

## Способ 3: Исправить масштабирование фона

Если элементы сбиваются при изменении размера экрана:

### Вариант A: Фиксированное соотношение сторон

В `css/style.css` найди `.background` и измени:

```css
.background {
    background-size: contain; /* вместо cover */
    aspect-ratio: 16 / 9; /* соотношение твоего фона */
}
```

### Вариант B: Единый масштабируемый контейнер

1. Узнай размер исходного изображения `lab-background.png` (например, 1920×1080)
2. Добавь в `:root`:
```css
:root {
    --bg-original-width: 1920px;
    --bg-original-height: 1080px;
}
```
3. Используй эти значения для позиционирования элементов

---

## Рекомендация

**Используй Способ 1** - он самый быстрый и точный. Скрипт автоматически вычислит все позиции.

---

## Пример использования

```javascript
// В консоли браузера:
positionHelper.setBgSize(1920, 1080); // Установить размер фона
positionHelper.start(); // Начать режим позиционирования
// Кликай на элементы...
positionHelper.stop(); // Остановить
```
